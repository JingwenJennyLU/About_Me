This is a programming project from The University of Chicago MACS 30121 (Designed by Dr. Zhao Wang)

-------------------------

These are the files for P1.

- sir.py: Skeleton code. I wrote my code in this file.

- test_sir.py: The automated tests. Use "py.text -xvk 'FUNCTION_NAME_PART'" to test.

- conftest.py: pytest configuration file

- pytest.ini and .pylintrc: Configuration files that can be safely ignored.

- grader.py: The grader script that will report score on the tests

- tests/: A directory containing files specifying the tests.

- sample_cities/: A directory with a few sample city files.

- README.txt: This file.
